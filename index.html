Styling the Background

Whatever you want to have a background color{
background-color: color here;
}
If you want the whole thing to have that color do
body{
background-color: color here;
}
You can do these things to do a background:
- For a solid color: rgb(#, #, #); or just say a color
- For a gradient: background: linear-gradient(to bottom, color 0%, another color 100%);

Borders

For borders you do this:
border: what_type_of_border #px color;(example: border: solid 4px pink)
Types of borders are:
-Solid
-Dotted
-Dashed
-Double
-Groove
-Ridge
-Inset
-Outset
-None
-Hidden
What side to do border:
You can choose what side to have the border on, in CSS you do this to decide:
- border-left: border_type #px color (for the left border)
- border-right: border_type #px color (for the right border)
- border-bottom: border_type #px color (for the bottom border)
- boreder-top: border_type #px color (for the top border)
